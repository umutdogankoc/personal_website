Production Tracker

Production Tracker is an ASP.NET Web Application, which provides operators to track their own production orders online and see the all of their work queue. Each operator is defined as a Dimension Value in Personal Dimension, the operator can log-in system with provided Username and Password. Production tracker has a feature to post order "temporarily",which indicates real-time finished quantity until the real NAV order will be posted by Full-User. After posting the real order, the operator screen will be updated with real quantities.
