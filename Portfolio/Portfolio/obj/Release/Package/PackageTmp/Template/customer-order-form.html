Customer Portal

Customer Portal is a ASP.NET Web application which builded with MVC with using microservices and OOP structure. Each customer has a password section in NAV(Customer Card), and they will be able to access customer portal once an Username and Password provided by authorized user. Customer can create a Sales Quote by in NAV by ordering from Web Application, check open sales quotes,orders,shipments,invoices and all open order status from left hand side menu.
